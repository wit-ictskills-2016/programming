Understanding class definitions

In this presentation we further explore further elements of classes and the Java language including how source code is written and presented.

Ticket machine

Here we shall examine some of the components of the ticket machine program. We shall look at some further elements of the Java language such as local variables and scope. We shall also introduce conditional statements, which offer choices between different actions as the program executes.

Ticket-Machine

In this lab we shall build a very basic virtual ticket machine and in doing so gain further experience with some of the elements of classes such as fields, constructors and methods. We shall conduct a rudimentary analysis of the ticket machine class and implement some design improvements.
